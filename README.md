# Infrastructure Projects

## Overview
A collection of hands-on infrastructure and network engineering projects demonstrating **network scanning, monitoring, and automation**. These projects showcase practical skills in **network security, system observability, and infrastructure management** — exactly the kinds of tasks performed in data center and network operations.

---

## Projects

### 1. Network Scanner & Asset Discovery Tool
- **Description:** Python-based scanner that discovers active devices on a subnet and identifies open ports (SSH, HTTP, HTTPS).  
- **Skills demonstrated:** Network discovery, automation, infrastructure security.  
- **Technologies:** Python, Nmap, Linux.  

### 2. Server Health Monitoring Dashboard
- **Description:** Prometheus and Grafana dashboard that monitors Linux servers’ CPU, memory, disk usage, and network traffic in real time.  
- **Skills demonstrated:** System observability, Linux administration, infrastructure monitoring.  
- **Technologies:** Prometheus, Grafana, Linux.  

### 3. Automated CI/CD Pipeline (Optional Future Project)
- **Description:** Automates deployment of a web service using Docker and GitHub Actions.  
- **Skills demonstrated:** Automation, CI/CD pipelines, containerization.  
- **Technologies:** Docker, GitHub Actions, Linux.  

---

## Why This Repository Stands Out
These projects are designed to **showcase practical, real-world infrastructure engineering skills**. They demonstrate the ability to **scan networks, monitor servers, and automate deployments**, which are highly valued in data center, networking, and cybersecurity roles.  

---

**Note:** All projects are safe to run on networks and servers you own or have permission to access.
